# Documentación del Proyecto

Este directorio contiene referencias, métricas y aprendizajes del proyecto **Serverless Inteligente (ABP 8)**.

- **architecture/**: diagrama ASCII y descripción de capas.
- **src/**: estructura simulada de funciones Lambda.
- **template.yaml**: plantilla AWS SAM con recursos representativos.
- **assets/**: capturas o material visual complementario.

El proyecto se diseñó como simulación técnica académica con foco en buenas prácticas cloud.
